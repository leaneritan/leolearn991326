# 📚 English Grammar Learning App

An interactive web-based application for learning and mastering English grammar through structured lessons, interactive exercises, and quizzes.

## 🎯 Features

- **📖 Comprehensive Lessons** - 6 structured grammar topics:
  - Parts of Speech
  - Tenses
  - Subject-Verb Agreement
  - Pronouns
  - Articles & Prepositions
  - Sentence Structure

- **✏️ Interactive Exercises** - Multiple exercise sets with:
  - Multiple choice questions
  - Fill-in-the-blank exercises
  - Instant feedback and explanations
  - Score tracking

- **🧪 Quiz Mode** - Test your knowledge with randomized questions

- **📊 Progress Tracking** - Monitor your learning:
  - Lessons completed
  - Exercises attempted
  - Quiz scores
  - Learning statistics

- **💾 Local Progress Saving** - Your progress is saved in your browser

## 🚀 How to Use

### Option 1: Online (Recommended)
1. Go to: `https://yourusername.github.io/Basic-English-Grammar-Book/`
2. Start learning immediately!

### Option 2: Local Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/leaneritan/Basic-English-Grammar-Book.git
   cd Basic-English-Grammar-Book
   git checkout learning-app
   ```

2. Open `index.html` in your web browser:
   - Double-click `index.html`, or
   - Use a local server:
     ```bash
     python -m http.server 8000
     # or
     python3 -m http.server 8000
     ```
   - Then visit: `http://localhost:8000`
